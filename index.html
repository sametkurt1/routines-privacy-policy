Privacy Policy

Routines values user privacy.

This application does not collect, store, or directly share users' personal data with third parties. The application does not require any account creation or server connection to perform its core functions.

Data Collection

  • The application does not collect personal data.
  • There is no account creation or login system.
  • No data is transmitted to the application developer.

Local Data Storage

  • All data within the application is stored only on the user's own device (locally).
  • This data cannot be accessed by the application developer.
  • When the application is deleted, all data stored on the device is also deleted.

Third Parties and Advertisements

The application may display rewarded ads (watch ads) to support the developer, subject to the user's request.

These advertisements are provided by third-party ad providers (e.g., Google AdMob). Viewing advertisements is entirely optional and is not mandatory for the core functions of the application.

The application developer does not collect or store any personal data through these advertisements. However, ad providers may process limited device information (e.g., Advertising ID) within the scope of their own privacy policies.

Children's Privacy

The application does not target children under the age of 13 and does not knowingly collect data from this age group. Advertisements are only displayed with the user's explicit consent.

Changes

This privacy policy may be updated from time to time. Any changes made will be published on this page.
